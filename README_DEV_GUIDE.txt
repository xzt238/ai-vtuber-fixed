================================================================================
                           1. 项目概述
================================================================================

咕咕嘎嘎是一个功能丰富的 AI 虚拟形象系统，支持：
- 实时语音对话（ASR 语音识别）
- 文字聊天（多 LLM 后端支持）
- 声音克隆训练（GPT-SoVITS v3）
- Live2D 虚拟形象
- 三层记忆系统（工作/情景/语义）
- 视觉理解（OCR + 图片理解）
